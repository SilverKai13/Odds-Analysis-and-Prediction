# Odds Analysis and Prediction

This project performs exploratory analysis and basic predictive evaluation on historical sports betting data. It provides insights into trends across different sports and assesses prediction accuracy based on bookmaker odds and match outcomes.

## Features

- Data cleaning and preprocessing
- Visual analysis of match distributions by sport
- Accuracy calculation of predicted vs. actual outcomes
- Easy-to-follow notebook format

## Dataset

The dataset used is `sports_predictive_analysis.csv`, which includes:

- Match results and outcomes
- Bookmaker odds
- Predicted vs. actual winner columns
- Sport category and metadata

## Usage

1. Clone the repository
2. Place the dataset `sports_predictive_analysis.csv` in the root directory
3. Run the notebook:

```bash
jupyter notebook oddsAI.ipynb
